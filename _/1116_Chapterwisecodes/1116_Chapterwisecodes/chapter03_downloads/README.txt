Alfresco Book - Chapter 3 Downloads
===================================

Files listed in the folder are used in the chapter as sample scripts and files.

Use the files as described in the Chapter 3 of the book.


Instructions to replace the default Logos:

- Go to tomcat\webapps\alfresco\images\logo folder

- Rename AlfrescoFadedBG.png, AlfrescoLogo32.png and AlfrescoLogo200.png files

- Copy all the three images from this folder 

	AlfrescoFadedBG.png, 
	AlfrescoLogo32.png,
and	AlfrescoLogo200.png
to tomcat\webapps\alfresco\images\logo folder
